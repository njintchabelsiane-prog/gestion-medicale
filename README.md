# 🏥 Gestion Médicale Patients

Application de bureau pour la gestion d'un cabinet médical : gestion des patients, des médecins et des consultations avec authentification sécurisée.

Projet réalisé dans le cadre du **Titre Professionnel Concepteur Développeur d'Applications (CDA)** — Green Up Academy.

---

## 🚀 Fonctionnalités

- 🔐 **Authentification sécurisée** avec hachage BCrypt
- 📊 **Tableau de bord** avec statistiques en temps réel
- 👥 **Gestion des patients** (CRUD complet + recherche)
- 🩺 **Gestion des médecins** (CRUD complet)
- 📋 **Gestion des consultations** (CRUD avec relations patient/médecin)
- 🎨 **Interface moderne** JavaFX avec design médical professionnel

---

## 🛠️ Technologies utilisées

| Couche | Technologie |
|--------|-------------|
| Langage | Java 17 |
| Interface | JavaFX 17 + FXML |
| Base de données | MySQL 8 |
| Accès données | JDBC + DAO Pattern |
| Sécurité | BCrypt (jBCrypt 0.4) |
| Build | Maven |
| IDE | Visual Studio Code |
